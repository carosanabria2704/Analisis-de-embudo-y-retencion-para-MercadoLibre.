Informe ejecutivo (C → F → I)
Prepare un Informe ejecutivo con formato C → F → I contestando a cada una de las preguntas del Negocio :
Contexto: Qué proceso se analizó y qué periodo abarca.
Hallazgos: Señalar los cohortes con las mejores y peores retenciones.
Implicaciones: Acciones concretas (p. ej. optimizar onboarding, notificaciones o recompensas).
Reflexión personal:
¿Qué etapa mejoraría primero?
¿Qué se evidenció sobre el comportamiento del usuario?
