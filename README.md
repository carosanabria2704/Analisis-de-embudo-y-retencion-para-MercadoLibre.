Informe ejecutivo (C → F → I)

Prepare un Informe ejecutivo con formato C → F → I contestando a cada una de las preguntas del Negocio :

1. Contexto: Qué proceso se analizó y qué periodo abarca.
2. Hallazgos: Señalar los cohortes con las mejores y peores retenciones.
3. Implicaciones: Acciones concretas (p. ej. optimizar onboarding, notificaciones o recompensas).

4. Reflexión personal:
¿Qué etapa mejoraría primero?
¿Qué se evidenció sobre el comportamiento del usuario?
